# editRequest - модуль, содердащий логику для изменения Профиля пользователя
    * editProfileImage - изменение изображения профиля Пользователя
    * editProfilePassword - изменение пароля профиля Пользователя
    * editProfileName - изменение отображаемого имени профиля Пользователя
    * editProfileEmail - изменение почты, привязанной к профилю Пользователя
